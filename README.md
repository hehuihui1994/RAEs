# RAEs
递归编码（Recursive auto-encoders, RAEs）。将单词表示为词向量，依据词向量得到某一段文字的向量空间，
然后逐层向上分析，继而得到整段文字的向量表示.<br>
## 优点
* 层次化的方法去理解结构和语意
* 可以用无标记或者有标记的数据进行训练，不需要任何语言字典
* 有利于分析较复杂的有联系的情感
